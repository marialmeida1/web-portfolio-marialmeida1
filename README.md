
# Portfólio - Mariana Almeida

## Descrição

Este é o portfolio digital da Mariana Almeida, destinado aos projetos desenvolvidos por mim na área de design e desenvolvimento. O site tem como objetivo apresentar minha trajetória e habilidades por meio de projetos que envolvem design e programação.

## Características de Implementação

- **Usando Next.js 15**: O projeto foi desenvolvido com a versão mais recente do Next.js, aproveitando os recursos de renderização estática e otimização de desempenho.
- **Consumo do Feed do Medium**: A aplicação consome o feed do Medium para exibir os posts mais recentes em formato de blog.
- **Next int**: Tradução entre os idiomas inglês e português utilizando o Next.js com a biblioteca `next-i18next`.
- **TypeScript**: O projeto foi desenvolvido utilizando TypeScript, garantindo maior robustez e segurança no código.

## Organização de Pastas

A estrutura de pastas do projeto é organizada da seguinte forma:

```
├── messages
│   ├── en.json
│   └── pt.json
├── public
│   ├── img
│   └── files
├── src
│   ├── app
│   │   ├── locale
│   │   │   └── (routes)
│   │   └── api
│   ├── components
│   ├── i18n
│   ├── lib
│   ├── types
│   ├── utils
│   └── middleware.ts
```

- **messages**: Contém os arquivos de tradução para os idiomas suportados (inglês e português).
- **public**: Armazena imagens e arquivos estáticos que são utilizados pelo site.
- **src**: Diretório principal onde está o código-fonte, organizado por funcionalidade. Inclui componentes, internacionalização, utilitários e middleware.

## Como usar

1. Clone este repositório em sua máquina local:
    ```bash
    git clone https://github.com/marialmeida1/web-portfolio-marialmeida1.git
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd portfolio
    ```
3. Instale as dependências:
    ```bash
    npm install
    ```
4. Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
5. Acesse o site no navegador em: [http://localhost:3000](http://localhost:3000).
